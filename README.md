# 🍲 Sabor Local

Aplicativo mobile de delivery desenvolvido no **FlutterFlow**, com foco na comercialização de pratos típicos da culinária mineira. O projeto foi criado como parte de uma atividade acadêmica, buscando aplicar conceitos de desenvolvimento mobile, integração com backend e consumo de APIs.

---

# 📖 Sobre o Projeto

O **Sabor Local** foi desenvolvido com o objetivo de oferecer uma plataforma intuitiva para pedidos de comida típica mineira. A aplicação permite que os usuários naveguem pelo catálogo de pratos, visualizem detalhes, adicionem itens ao carrinho e realizem pedidos de forma simples e organizada.

O backend foi desenvolvido utilizando **Xano**, responsável pelo gerenciamento do banco de dados, autenticação de usuários e disponibilização das APIs REST consumidas pela aplicação.

Antes da implementação, as interfaces foram planejadas e prototipadas utilizando **Figma**, permitindo validar o fluxo de navegação e a experiência do usuário.

---

# 🚀 Funcionalidades

- Cadastro de usuários
- Login e autenticação
- Catálogo de pratos típicos da culinária mineira
- Visualização dos detalhes dos pratos
- Carrinho de compras
- Realização de pedidos
- Integração com APIs REST
- Gerenciamento de dados utilizando Xano
